## 🧱 PRODUSUL SOIA — ARHITECTURA

SOIA are 4 niveluri tehnice + o interfață finală:

| Nivel | Denumire | Ce face |
|-------|----------|---------|
| 1 | CONTEXT | Memoria firmei: SOP-uri, valori, produse, strategie |
| 2 | DATE | Integrări: Google Sheets, Stripe, CRM, Analytics |
| 3 | INTELIGENȚĂ | Conversații: ședințe, Slack, emailuri procesate |
| 4 | AUTOMATIZARE | Execuție: Make.com, Claude Code, scripturi 24/7 |
| ★ | INTERFAȚĂ | Daily Brief Telegram + Dashboard web |

### Etapele de livrare (cum vindem):

| Etapă | Niveluri | Implementare | Retainer/lună |
|-------|----------|-------------|---------------|
| SOIA CONTEXT (Etapa 1) | Nivel 1 + Interfață | 250-400$ | 100-180$ |
| SOIA DATA (Etapa 2) | Nivel 2 + Nivel 3 | 350-600$ | 150-280$ |
| SOIA AUTO (Etapa 3) | Nivel 4 + Control complet | 400-700$ | 200-350$ |
| SOIA COMPLET | Toate 4 niveluri | 1.000-1.700$ | 450-810$ |

### KPI-urile de succes pentru client:
- **Autonomia față de birou:** poate conduce firma de pe telefon?
- **% taskuri automate:** ținta 60-70% din taskurile zilnice
- **Venit per angajat:** crește fără a mări echipa
