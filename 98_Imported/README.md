# Imported Context

Acest folder pastreaza materiale importate ca sursa bruta sau legacy, pentru a nu pierde context util in timpul reorganizarii vault-ului.

## Ce intra aici

- vechiul `klaryoflow-context`
- materiale brute importate temporar
- note care nu au fost inca curatate in structura principala

## Regula

Materialele din acest folder nu sunt structura finala.

Ele sunt:

- sursa
- backup
- referinta

## Cum lucram cu acest folder

- nu editam aici continutul strategic final, decat pentru etichetare minima
- extragem ideile utile in structura principala a vault-ului
- pastram acest folder pentru trasabilitate si context istoric

Structura principala de lucru ramane:

- `01_Brand`
- `02_Produs`
- `03_Go_To_Market`
- `04_Operare`
- `05_Clienti`
- `06_Templates`
