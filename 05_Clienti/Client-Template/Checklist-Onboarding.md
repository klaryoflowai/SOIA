# Checklist Onboarding Client

## Inainte de kickoff

- [ ] exista acord clar pe oferta
- [ ] exista owner principal la client
- [ ] exista obiectiv clar pentru primele 7 zile
- [ ] exista calendar pentru interviu

## In timpul kickoff-ului

- [ ] intelegere business model
- [ ] intelegere echipa si roluri
- [ ] intelegere proceselor care blocheaza fondatorul
- [ ] inventariere tool-uri
- [ ] definire KPI minim

## Pentru SOIA CONTEXT

- [ ] colectat material brut de context
- [ ] definit formatul Daily Brief
- [ ] ales Quick Win-ul
- [ ] explicat cum va arata livrarea

## Pentru primele 7 zile

- [ ] owner client disponibil pentru feedback
- [ ] o revizie intermediara programata
- [ ] validare finala programata

## Dupa livrare

- [ ] feedback textual primit
- [ ] KPI initial comparat cu rezultatul
- [ ] potential de studiu de caz evaluat
- [ ] pas urmator propus
