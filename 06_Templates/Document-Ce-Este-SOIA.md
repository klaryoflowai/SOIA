# Ce este SOIA

SOIA este un sistem operational cu inteligenta artificiala pentru antreprenorii care simt ca firma depinde prea mult de ei.

Nu este doar un chatbot.
Nu este doar o automatizare.
Nu este doar un dashboard.

SOIA pune impreuna:

- contextul firmei
- cifrele importante
- conversatiile si deciziile
- executia automata
- o interfata zilnica de control

## Ce primeste antreprenorul

- claritate dimineata
- control de pe telefon
- mai putina munca administrativa
- mai putine blocaje care ajung la el

## Cum se instaleaza

In 3 etape:

1. SOIA CONTEXT
2. SOIA DATA
3. SOIA AUTO

Clientul poate incepe cu prima etapa si obtine deja valoare din prima saptamana.

## Cui i se potriveste

- firme 10-50 angajati
- fondatori supraincarcati operational
- echipe care au tool-uri, dar nu au claritate

## Promisiunea

Conduci firma. Nu firma pe tine.
