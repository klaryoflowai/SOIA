# Arhitectura SOIA

SOIA nu este doar un software. Este un sistem operational construit in jurul firmei.

## Cele 4 niveluri

| Nivel | Denumire | Rol | Exemple |
|------|----------|-----|---------|
| 1 | Context | Memoria permanenta a firmei | SOP-uri, valori, produse, organigrama, strategie |
| 2 | Date | Strat cantitativ si operational | Sheets, CRM, Stripe, analytics, rapoarte |
| 3 | Inteligenta | Strat calitativ si relational | sedinte, Slack, emailuri, feedback, decizii |
| 4 | Automatizare | Strat de executie | Make, n8n, Claude Code, scripturi, remindere |
| * | Interfata | Ce vede antreprenorul | Telegram, Daily Brief, dashboard web |

## Explicarea pe intelesul clientului

### Nivelul 1 - Context

"Este ca un consultant care a citit deja toate documentele firmei tale inainte de prima zi de munca."

### Nivelul 2 - Date

"Este tabloul de bord al masinii, dar unul care iti si explica de ce s-a aprins becul din bord."

### Nivelul 3 - Inteligenta

"Este asistentul care a fost prezent la toate sedintele si stie ce s-a decis."

### Nivelul 4 - Automatizare

"Este o echipa de executie care face taskurile repetitive 24/7 dupa regulile tale."

### Interfata

"Este locul din care vezi totul si comanzi totul, direct de pe telefon sau din dashboard."

## De la tehnic la comercial

| Etapa comerciala | Niveluri | Ce primeste clientul |
|------------------|----------|----------------------|
| SOIA CONTEXT | Context + Interfata | AI care cunoaste firma + Daily Brief |
| SOIA DATA | Date + Inteligenta | cifre, semnale si observatii unificate |
| SOIA AUTO | Automatizare + control | taskuri executate si orchestrat automat |
| SOIA COMPLET | toate | sistem operational AI complet |

## KPI-urile de succes

### 1. Autonomie fata de birou

Poate antreprenorul conduce businessul doar cu telefonul?

### 2. Procentul de taskuri automate

Din 100 de taskuri repetitive, cate au iesit din operarea manuala?

### 3. Venit per angajat

Cresterea companiei se intampla fara crestere proportionala a echipei?

## Ce este livrarea minima credibila

Pentru primul contact comercial, produsul trebuie sa para foarte concret. De aceea SOIA trebuie explicat ca:

- memorie + claritate
- date + context
- actiune + control

Nu intra in detalii tehnice daca nu sunt cerute.
